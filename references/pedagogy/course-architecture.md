# 课程结构（共同教学法）

> 何时阅读：设计课程结构时阅读。所有课程都适用。

## 节数

- 默认 6-8 节，可按素材密度调整。
- 最后一节做总复习或综合应用，不引入全新主题。
- 不要为一般性请求交付一个长页面。

## 课程级

- 一个聚焦的课程主题
- 清晰的前置要求
- 递进的课程顺序
- 课程标题
- 2-3 句课程描述
- 预估学习时长（`duration`）

## 每节结构

每节都包含：`title` / `goal`（一句话）/ `concepts`（2-3 项，总复习为 `[]`）/ `objectives`（2-4 项）/ `body` / `flashcards` / `quiz` / `sources`。

- `id` 从 1 开始连续递增。
- 最大 `id` 自动识别为总复习，使用累积评分。
- 保持紧凑，确保预览响应流畅：不要嵌入巨量文章或数据块。
